Multi-Model AI Classifier

Powered by TensorFlow & Keras
Supporting MobileNetV2, ResNet50, EfficientNetB0 & DenseNet121

How to Use the Multi-Model Image Classifier

  1. Create class folders by specifying the number of categories
  2. Name each folder according to your classes (e.g., "cats", "dogs")
  3. Upload training images to respective folders
  4. Ensure each class has at least 10-20 images for good results

  • MobileNetV2: Fast, lightweight model ideal for mobile deployment
  • ResNet50: Deep network with residual connections, good accuracy
  • EfficientNetB0: Balanced efficiency and performance
  • DenseNet121: Dense connectivity, excellent feature reuse

Training typically takes 5-15 minutes per model depending on dataset size and hardware.

Once models are trained:

  1. Navigate to the Predictions tab
  2. Upload an image for classification
  3. View predictions from all trained models
  4. Compare confidence scores and ensemble results
